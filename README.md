##O Projeto:
O projeto é um jogo de milhão que foi desenvolvido para o Cesar. Porem é escalavel para que mais pessoas possam utilizar nosso sistema.

##O que faz?
O sistema é composto por um backend que é responsável por gerenciar as requisições e respostas do frontend, e um frontend que é responsável por mostrar as perguntas e opções de resposta para o usuário.

##Tecnologias Utilizadas:
- Node.js
- TypeScript
- Json Web Token
- Bcrypt
- Express
- mongoDB
- Git
- GitHub

## Como usar:
- Clone o repositório
- Instale as dependências
- Crie um arquivo .env
- Inicie o servidor
- Acesse o frontend
